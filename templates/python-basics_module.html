Python Basics

Python is a versatile and easy-to-learn programming language. This module will introduce you to its core concepts, allowing you to write your first programs.

1. Variables and Data Types

In Python, a **variable** is a container that stores data. There are different kinds of data, called **data types**, such as `strings` (text), `integers` (whole numbers), and `booleans` (True/False).

2. Control Flow

Control flow allows you to tell your program what decisions to make. The most common tools are **if/elif/else statements**, which run code based on a condition, and **loops** (like `for` and `while`), which repeat a block of code.

3. Functions

A **function** is a reusable block of code that performs a specific task. Defining your own functions helps you organize your code and avoid repeating yourself. Python also has many built-in functions, like `print()` to display output.

4. Data Structures

Python provides built-in **data structures** to organize your data. The most common are **lists** (ordered collections of items) and **dictionaries** (collections of key-value pairs).
